## Why Ruby?

Ruby is designed for programmer happiness and productivity. Its elegant syntax is natural to read and easy to write.

```ruby
# This is Ruby code
def greet(name)
  puts "Hello, #{name}!"
end

greet("Ruby Learner")
```
