Chapter 39 – Object Serialization (Marshal) Beginner

Persist Ruby objects by dumping and loading them with the built-in `Marshal` module.

> Adapted from Satish Talim's object serialization lesson. Ruby's Marshal module converts objects to byte streams (serialization) and back (deserialization). Handy for caching, storing session data, or quick persistence.

Dumping and loading

ruby.rb 

data = { name: "Satish", skills: %w[ruby rails] }

File.open("data.dump", "wb") do |file|
  Marshal.dump(data, file)
end

loaded = File.open("data.dump", "rb") { |file| Marshal.load(file) }
puts loaded == data  #=> true
You can also dump to a string:
ruby.rb 

payload = Marshal.dump(data)
restored = Marshal.load(payload)

Unsupported objects

Some things can't be marshaled (e.g., procs, bindings, IO objects, singleton objects). Catch TypeError if you're unsure.
