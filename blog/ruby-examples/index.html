Practise alongside the Ruby tutorials

The dedicated Ruby tutorials page now houses the full learning path. Continue with Chapter 2 for flow control and collections or Chapter 3 for methods and blocks. If you’d like a quick refresher before diving back in, paste these examples into IRB and modify them after working through the practice checklists.
ruby.rb 
# A simple Ruby class
class Person
attr_accessor :name, :age
def initialize(name, age)
@name = name
@age = age
end
def greet
"Hello, my name is #{@name} and I am #{@age} years old."
end
end# Create a new person
person = Person.new("Alice", 30)
puts person.greet
